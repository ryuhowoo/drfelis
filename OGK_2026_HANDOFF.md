## 2. Firebase / 트래커 (현재 정상 작동)
- **전용 프로젝트**: `drfelis-ogk-tracker` (asia-southeast1). 옛 공유 프로젝트 `drfelis-refreshteam`에서 분리함.
- **RTDB 규칙**: 공개 `{ "rules": { ".read": true, ".write": true } }` (무인증, 누구나 작성/수정/삭제 공유 저장).
- **경로**: `drfelis-2026-tracker/state`. `FB_CONFIG`는 HTML 내 `const FB_CONFIG`.
- 동기화 배지: 실시간 동기화 / 저장 중 / **저장 안 됨 — DB 규칙 확인**(권한거부) / 오프라인(로컬).
- 데이터 모델: `defaultData()` / `mergeData()` / `saveData()`. 새 키 추가 시 두 함수 모두 갱신해야 동기화에 살아남음(예: `gateVals`).
- 구글 로그인 게이트는 **제거됨**(원하면 @drfelis.com 게이트 재부착 가능).

### 트래커 게이트(중요 기능)
`GATE_DEFS`가 메트릭 구조: 각 게이트의 **목표값마다 실적 입력칸** + **자동 판정**(전부 충족→✓, 미달→✕, 미입력→○).
- g1 공헌이익액+30% 방어 = **공헌이익률 ≥30%** + **공식몰 비중 ≥37/38/40%**(2분기/3/4)
- g2 **구독 내 영양 매출 비중** ≥26/28/30% (라벨 명확화 완료 — "영양 매출 비중"만으로는 구독 내/전체 혼동되어 명시) · g3 광고비/매출 **≤12%** · g4 브랜드 캠페인 누적 ≥1/2/3회
- 값은 `D.gateVals[q][gid][metricKey]`, 자동 상태는 `computeGateState()`.

## 3. 핵심 숫자 (정합 기준 — 바꾸면 연쇄 갱신 필요)
- B2C 전체: **2024 65.37억 → 2025 89.21억(+36.5%)**.
- **상반기 1~5월 40.15억 확정(+30.1%)**. 작년 6~12월 base **58.35억**.
- **잔여 매출 브리지(s2 핵심)** — 연매출 = 40.15(확정) + 잔여(6~12월):
  | 시나리오 | 작년기반×성장 | +신제품 | +재고회복 | 잔여 | 연매출 | 달성 | YoY |
  |---|---|---|---|---|---|---|---|
  | 보수(라인 -5%) | 55.4 | +10.6 | +1.85 | 67.9 | **108.0억** | 90% | +21% |
  | **기준선(중간,+15%)** | 67.5 | +10.6 | +1.85 | 80.0 | **120.1억** | 100% | +34.6% |
  | 실측추세(+25%) | 72.9 | +10.6 | +1.85 | 85.4 | **125.5억** | 105% | +41% |
  - **중간 기준 = 120억 앵커**(분기 빌드업 합계 120.1과 일치). 신제품·재고는 고정, 차이는 기존 라인 유기성장.
  - 신제품 +10.6 = 카사바·오메가3·유산균(Q3 7.51 + Q4 3.09).
  - **재고회복**: 품절 기회비용 **실측 4.08억**(모래2.27·영양케어1.42·간식0.37·유산균0.02·용품0.01) 중 **보수적 ~1.85억(약 45%)만 1회성 회수** 반영. **완전 재고는 8월부터**.
- **분기 빌드업**: Q1 23.62(실측)·Q2 ~27.6·Q3 35.64·Q4 33.22 = 120.1.
- **월 단위 현실 점검 카드**: 역대 최고월 **9.5억 = '25.11월**(꽃같은 3주년 최대할인 + 유산균 런칭 + 연말 소비심리 피크, 단일 피크). 기준선은 하반기 월평균 11.4억(기록 +20%) → "신제품 라인·구독 1.4억/월 바닥·4주년(10월)+블프(11월) 분산·상반기+30% 절반" 근거로 방어. 톤: **보수 108=바닥, 120=목표**.

## 4. 지표 정합(P1, 완료)
- 구독 게이트 주지표 = **구독 내 영양 매출 비중**(26→28→30%, g2). "교차구독"은 그 비중을 올리는 수단(단어 유지).
- **일반 판매(비구독) 영양 비중**은 별도 보조 KPI로 관리 — 게이트에는 안 들어감(액션→결과 거리가 멀어 게이트 부적합). 광고/번들 결과로 추적.
- **공식몰 매출 비중 37→40%**(게이트 4Q 40%와 정합). 35%는 **스스 영양 SKU 비중**(다른 지표, 라벨로 구분) + 구조전환 기회비용 블록 1곳만.
- s3 KPI: 구독자 **3,357**(결제중 2,991+일시정지 366) · 1회 평균 **~41,900** · 월 구독매출 **~1.40억** · 공식몰 매출 중 구독비중 **32.98%**.
- 구독 구성(결제중 2,991): 모래만 **2,033(68.0%)** / 모래+영양 **273(9.1%)** / 영양만 **685(22.9%)**.
- "영양 55.3% vs 모래 30.9%" = **공헌이익률(마진율)** — 영양이 1.8배 고마진 → 구독 내 영양비중↑ = 구독마진↑.
