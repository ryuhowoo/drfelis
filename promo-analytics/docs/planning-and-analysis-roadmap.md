# 캠페인 플래닝 · 성과 비교분석 고도화 로드맵

> 목적: "가장 정교하게 만들어 단순하게 보이도록" — MD가 노가다 없이 빠르게 신뢰할 수 있는 플랜을
> 짜고, 종료 후 성과를 한 번에 비교·학습하게 한다. 이 문서는 다음 작업의 기준점이다.

## 1. 이커머스/리테일은 어떻게 하나 (업계 관행)

- **프로모 캘린더 + 그리드**: 분기/월 단위로 행사를 한 판에 깔고(채널·기간·혜택·목표), 충돌·공백을
  먼저 본다. (대형사는 Anaplan·o9·RELEX·Nextail 같은 플래닝 툴, 중견은 대부분 Excel/Sheets 템플릿)
- **SKU 단위 마크다운 플랜 + 마진 가드레일**: 품목별 할인 깊이를 정하되 **최소 공헌이익률** 같은
  가드레일을 둔다. 가드 위반은 '막기(gate)'보다 '경고(guardrail)'로.
- **수요 예측 = baseline + lift**: 과거 유사 행사(혜택유형 × 시즌 × 할인깊이)의 증분율을 baseline에
  얹어 예측, 신뢰도(표본·일관성)를 함께 표기. (우리 predict 엔진과 동일 철학)
- **사후 분석 = 증분 + 달성 + 셀스루**: 단순 매출이 아니라 **증분(incrementality, vs baseline)**,
  계획 대비 달성률, 셀스루(재고 소진), 구독 분리, 반품률, 프로모 ROI(=증분이익/투입).
- **핵심 지표 표준**: GMV/매출 · 판매수량 · AOV · 공헌이익/률 · 증분(±CI) · 후광(halo) ·
  셀스루 · 마크다운 깊이 · 프로모 ROI. (대부분 이미 구현됨)

## 2. 우리가 채택할 효과적 기준 (현 상태 → 보강)

| 기준 | 현재 | 보강 |
|---|---|---|
| 플랜 = 옵션(묶음) + 마진 가드레일 | 옵션·공헌 계산 O | **최소 공헌이익률 경고**(가드레일) 추가 |
| 예측 = baseline+lift+신뢰도 | predict 엔진 O(증분/전체 분리·null) | 작성화면 임베드 O · 사후 정확도 루프 |
| 사후 = 증분·달성·구독분리 | O | **셀스루(재고 소진율)** 지표 추가 |
| 노가다 제거 | 서브상품 벤치마크 추천 O | **엑셀 왕복**(아래 3장) |
| 한눈에 보기 | 대시보드 | **프로모 캘린더** 뷰 |

## 3. 엑셀 브리지 — 가장 큰 리소스 절감 (사용자 제안 구체화)

MD는 결국 Excel에서 산다. 앱을 '단일 진실원천'으로 두되 **Excel과 왕복**시키면 학습비용 없이 속도가 난다.

### 3.1 플랜 → Excel 내보내기 (수식 내장)
- 완성 플랜을 **수식이 살아있는** xlsx로 내려받기. MD가 오프라인에서 단가/수량을 바꾸면
  옵션단가·할인율·공헌이익이 **자동 재계산**(앱과 동일 식).
- 시트 구성:
  - `플랜` 시트: 옵션 × SKU 행, `옵션단가 = Σ(수량×단가)`, `할인율 = 1−옵션단가/소비자가합`,
    `공헌 = (옵션단가×mult − Σ원가×수량)×예상수량` 수식 셀로.
  - `SKU마스터` 시트(참조용, read-only): 전 SKU의 코드·소비자가·상시가·원가. 플랜 시트는
    `XLOOKUP(코드, SKU마스터!...)` 로 가격·원가를 끌어와 **어긋남 0**.
  - `요약` 시트: 목적별 총합(매출/구매건수/판매수량/공헌·률) 수식.

### 3.2 Excel → 플랜 가져오기 (한 번에 완성)
- **같은 템플릿**으로 채운 xlsx를 올리면 옵션·SKU·수량·단가·쿠폰을 파싱해 **플랜 자동 생성**.
  SKU는 코드 우선 매칭(없으면 기초상품명) — 이미 만든 리치 파서/정규화 재사용.
- 업로드 시 **검증 리포트**: 미매칭 SKU·마진 가드 위반·수식 깨짐을 먼저 보여주고 확정.

### 3.3 왕복 무결성
- 내보낸 파일에 `plan_id`·버전·체크섬을 숨겨, 다시 올릴 때 **버전 충돌/덮어쓰기**를 안전 처리.
- 앱↔Excel 식이 **단일 출처**(lib/plan.ts ↔ confirm_plan SQL ↔ 엑셀 수식)로 일치하도록 테스트.

## 4. 단계별 실행안 (작은 단위로, 검증하며)

1. **플랜 Excel 내보내기**(수식+SKU마스터 시트) — 가장 빠른 가치, 위험 낮음.
2. **Excel 가져오기 → 플랜 자동 생성** + 검증 리포트(미매칭·가드 위반).
3. **마진 가드레일 경고**(최소 공헌이익률) — 플랜 에디터·확정 단계.
4. **프로모 캘린더 뷰**(캠페인 허브에 월별 그리드).
5. **셀스루 지표** + **예측 정확도 루프**(예측 vs 실제 자동 비교 → 상수 보정).
6. **카테고리 보강**(현재 16%) → 서브상품 카테고리 매칭 강화.

## 5. "정교하게 → 단순하게" 원칙 (UX 가드)

- **점진적 노출**: 기본은 추천값으로 채워 보여주고, 세부 조정은 펼쳐야 보이게.
- **가드레일 > 게이트**: 막지 말고 경고. 사용자가 판단하게.
- **단일 진실원천**: 앱이 원본, Excel은 왕복 도구. 식은 한 곳에서.
- **기본값은 데이터에서**: 벤치마크·유사 사례로 prefilled → 빈 칸에서 시작하지 않게.
- **모든 화면은 "지금 뭘 봐야 하나"에 답**: 성과 없으면 '성과 올리기'만, 있으면 달성률만.
